Binary convert()

To use on a page in the yaml page settings add scripts: and zachdyerdesign/binary.js and zachdyerdesign.string.js. Pass a binary value into the function and it returns a whole number value. If any other value is passed in it returns false.

{% highlight js linenos%} zachdyerdesign.binary.convert(101); // returns 5 {% endhighlight %}
